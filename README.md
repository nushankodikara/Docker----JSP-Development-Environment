# Why this project is a thing

JSP is a dying technology compared to modern web frameworks, however it is still used in many projects, specially in schools and universities. This project aims to provide a user friendly and easy to setup development environment for JSP projects using Docker.

# How to use

## Requirements

-   Docker
-   Docker Compose
-   Docker Desktop (optional)

## Setup

1. Clone this repository
2. Run `docker-compose up` in the project root
3. Open `localhost:8888/webapp` in your browser

## Development

1. Open the project in your favorite IDE
2. Make changes to the project in webapp folder
3. Refresh the browser to see the changes
